# Straw Hat Pirates Crew Registration

A SwiftUI iOS app replicating a dynamic crew registration form inspired by One Piece, plus a custom profile screen from Dribbble. Built for a college iOS development assignment to showcase layout mastery, form validation, and interactive UI components.[file:1][file:2]

## Features

**Part 1: Straw Hat Pirates Form**
- Real-time validation for pirate name, Den Den Mushi email, bounty ID (2-20 chars), and matching Devil Fruit powers [file:2]
- Interactive toggles for crew loyalty and treasure updates, plus bounty slider (50M-500M berries) with formatted display [file:2]
- One Piece-themed styling with custom colors, scrollable layout, and success alert that resets the form [file:2][file:1]

**Part 2: Profile Screen**
- Editable fields for personal email, school email, college, and secure password with show/hide toggle [file:4]
- Email validation, notifications toggle, and save button with confirmation alert [file:4]
- Clean card-based design adapted from Dribbble with profile image, circular avatar, and modern input styling [file:4]

## Tech Stack
- SwiftUI for declarative UI and state management
- ScrollView, VStack/HStack layouts, and precise padding/spacing
- Custom colors, monospaced fonts for bounty, and icon integrations [file:2][file:4]
